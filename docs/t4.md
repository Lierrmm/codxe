## Multiplayer

### Dvars

#### `noclip_brushes` _(string)_

A server-info dvar that controls which brushes have player collision disabled. It holds a space-separated list of brush indices.

Special values:

- `""` - Restores all brushes to their original collision state.
- `"*"` - Disables player collision on every brush in the clipmap.

```text
set noclip_brushes "";    // reset all brush collision
set noclip_brushes "*";   // disable all brush collision
set noclip_brushes "4 7"; // disable collision on brushes 4 and 7
```

### GSC Functions

#### `GetPlayerclipBrushesContainingPoint(vec3 <point>)`

Returns an array of playerclip brush indices whose bounds contain the given point.

Usage example:

```gsc
brushes = GetPlayerclipBrushesContainingPoint(self.origin);
```

### GSC Player Methods

#### `ButtonPressed(string <button>)`

Checks if a specific local key or button is currently pressed.

Usage example: `if (self ButtonPressed("BUTTON_A")) { ... }`

#### `SprintButtonPressed()`

Returns true if the sprint button is pressed.

Usage example: `if (self SprintButtonPressed()) { ... }`

#### `JumpButtonPressed()`

Returns true if the jump button is pressed.

Usage example: `if (self JumpButtonPressed()) { ... }`

#### `SetVelocity(vec3 <velocity>)`

Sets the player's velocity to the given 3D vector.

Usage example: `self SetVelocity((0, 0, 300));`

#### `GetForwardMove()`

Returns the player's raw forward/backward movement input.

Usage example: `forward = self GetForwardMove();`

#### `GetRightMove()`

Returns the player's raw left/right movement input.

Usage example: `right = self GetRightMove();`

#### `CloneBrushModelToScriptModel(entity <brushModelEnt>)`

Clones a brush model's collision model to a `script_model` entity.

Usage example:

```gsc
scriptModel CloneBrushModelToScriptModel(brushEntity);
```
